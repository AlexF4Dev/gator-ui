How to Manually Install web hook?

NOTE: You have to do this only if you see an error in installing webhooks.

1. Goto your organization page on the github
2. Navigate to the "Settings" tab.
3. Select the "Webhooks" option on the left menu.
4. Click "Add Webhook"
5. put 'https://gatorgithook.azurewebsites.net/api/httptrigger' in payload URL
6. Select "application/json" as the encoding type.
7. select radio button 'Let me select individual events.'
8. check 'Pull requests'
9. click on Add webhook
