How to Manually Install web hook?

You have to do this only if you get error in installing webhooks.
Goto your organization page on the github
Navigate to the "Settings" tab.
Select the "Webhooks" option on the left menu.
Click "Add Webhook"
put 'https://gatorgithook.azurewebsites.net/api/httptrigger' in payload URL
Select "application/json" as the encoding type.
select radio button 'Let me select individual events.'
check 'Pull requests'
'click on Add webhook'
