Help

Webhook installtion failed, because you may not have the permission. Please install it manually.

Someone else from your orgnization may have already installed the hook. We need only one hook per organization. If that is the case than you are good.

Without webhooks Git Gator will not update your data. Please ask the admin of your orgnization to install webhook.

Follow these simple steps to Manually Install web hook?

1. Goto your organization page on the github
2. Navigate to the "Settings" tab.
3. Select the "Webhooks" option on the left menu.
4. Click "Add Webhook"
5. put 'https://gatorgithook.azurewebsites.net/api/httptrigger' in payload URL
6. Select "application/json" as the encoding type.
7. select radio button 'Let me select individual events.'
8. check 'Pull requests'
9. click on Add webhook
